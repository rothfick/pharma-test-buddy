## ✨ Overview
Pharma Test Buddy is an AI-assisted quality assurance tool designed to enhance and optimize workflows within the pharmaceutical industry. It focuses on ensuring the integrity, validation, and traceability of critical processes, leveraging artificial intelligence to identify potential issues and streamline testing cycles.

## 🧱 Tech Stack
*   **TypeScript**: Provides strong typing and improved code quality.
*   **AI**: Powers intelligent analysis and assistance for QA tasks.
*   **Vite**: A fast build tool for modern web projects.
*   **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
*   **Vitest**: A blazing fast unit-test framework powered by Vite.

## 🚀 Features
*   **Domain QA**: Specialized testing capabilities tailored for pharmaceutical regulations and standards.
*   **Validation Assistance**: AI-driven support for validating complex workflow steps.
*   **Traceability Assurance**: Enhances tracking and auditing of testing processes and results.
*   **Intelligent Anomaly Detection**: Proactively identifies deviations and potential risks.

## 🛠️ Quickstart
To get started with Pharma Test Buddy, follow these steps:

```bash
npm install
npm run dev
```

## 🗺️ Project Structure
The `pharma-test-buddy` project is structured around a standard modern web application setup. Source code resides primarily in the `src` directory, with `components` housing reusable UI elements. Configuration files for build tools (`vite.config.ts`, `postcss.config.js`, `tailwind.config.ts`), linting (`eslint.config.js`), and TypeScript (`tsconfig.json` family) are at the root. Public assets are found in the `public` folder, and `supabase` contains backend-related configurations.
